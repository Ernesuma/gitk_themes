# Dark Dracula Theme
To use this theme on Linux, copy the 'gitk' file to  
`~/.config/git/`

You could use for example:  
`$ mkdir ~/.config/git/`  
`$ cp gitk ~/.config/git/`

# Thanks
The theme in this repository is originaly taken from the Dracula theme repository. It has been modified to be even darker.
